#  CyberSecurity Awareness Bot

A simple C# console chatbot that teaches users cybersecurity basics through interactive conversation, ASCII art, and audio.

---

##  Features

*  Chat-based interaction
*  Tips on passwords, phishing, and safe browsing
*  ASCII art logo display
*  Typing animation
*  Audio greeting

---

##  Usage

Ask questions like:

* "What is phishing?"
* "How do I create a strong password?"

Type `exit` to quit.

---

## Files

* `Program.cs` – Main entry + ASCII art
* `Chatbot.cs` – Chat logic
* `ResponseHandler.cs` – Answers
* `UIHelper.cs` – UI effects
* `AudioPlayer.cs` – Sound

---

##  Notes

Ensure `logo.jpg` and `greeting.wav` are in the project folder.
